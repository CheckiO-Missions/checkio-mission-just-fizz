Ok, what should I start from?

First of all, read about if-elif-else statement and use it in your code: 

if ...:
    ...
else:
    ...

Ok. What to do next?

You can check how a number is divisible by another one with the % (modulo) operator. 

if num % n == 0: # num is evenly divisible by n
    ...

The more pythonic way is to write as: 

if not num % n:
    ...

How exactly to use it in the mission?

In case, the num is evenly divisible by 3, the function should return "Fizz". So, let's write it: 

if not num % 3:
    return "Fizz"

How to write the other case?

In the other case (when num is not divisible by 3 evenly), the function should return num as string. If you need the string representation of a number, use str() built-in function.

So, how my final solution should look like?

You just need to wrap your code into function expression.

Here is the whole solution:

def checkio(num: int) -> str: 

    if not num % 3:
        return "Fizz"
    else:
        return str(num)
