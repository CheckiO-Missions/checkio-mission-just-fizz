这是一个简单的任务，解决的想法也很简单：创建一个单一的条件检查，如果条件是True ，就返回一个值，否则返回另一个值。

好吧，我应该从什么开始呢？

首先，阅读关于if-elif-else statement ，并在你的代码中使用它： 

if ...:
    ...
else:
    ...

好的。接下来要做什么？

你可以用% (modulo) operator ，检查一个数字是如何被另一个数字整除的。 

if num % n == 0: # num is evenly divisible by n
    ...

更为pythonic的方法是写成： 

if not num % n:
    ...

在任务中究竟如何使用？

如果num 能被3 偶数整除，该函数应该返回"Fizz" 。所以，我们来写一下： 

if not num % 3:
    return "Fizz"

如何写出其他情况？

在其他情况下（当num 不能被3 匀称地除以时），该函数应返回num 作为字符串。如果你需要一个数字的字符串表示，请使用str() 内置函数。

那么，我的最终解决方案应该是什么样子的呢？

你只需要把你的代码包装成函数表达式。

下面是整个解决方案：

def checkio(num: int) -> str: 

    if not num % 3:
        return "Fizz"
    else:
        return str(num)

当然，抓住它!

条件就写在返回语句里面。 

def checkio(num: int) -> str:
    
    return str(num) if num % 3 else 'Fizz'

案例被当作元组的元素，其中索引是modulo的布尔值。 

def checkio(num: int) -> str:

    return ("Fizz", str(num))[bool(num % 3)]
