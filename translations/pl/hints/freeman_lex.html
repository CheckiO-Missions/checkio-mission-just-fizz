Nie mam pojęcia, jak zacząć rozwiązywać tę misję.

jest to prosta misja z prostym pomysłem na rozwiązanie: proszę utworzyć pojedynczy warunek sprawdzający i zwrócić jedną wartość w przypadku, gdy warunek jest True i inną wartość w przeciwnym razie.

Ok, od czego powinienem zacząć?

Przede wszystkim proszę poczytać o if-elif-else statement i używać go w swoim kodzie: 

if ...:
    ...
else:
    ...

Ok. Co robić dalej?

Mogą Państwo sprawdzić, jak liczba jest podzielna przez inną za pomocą % (modulo) operator. 

if num % n == 0: # num is evenly divisible by n
    ...

Bardziej pythonicznym sposobem jest zapisanie jako: 

if not num % n:
    ...

Jak dokładnie wykorzystać to w misji?

W przypadku, gdy num jest podzielne przez 3, funkcja powinna zwrócić "Fizz". Więc proszę to napisać: 

if not num % 3:
    return "Fizz"

Jak napisać drugi przypadek?

W drugim przypadku (gdy num nie jest podzielne przez 3 równo), funkcja powinna zwrócić num jako ciąg znaków. Jeśli potrzebują Państwo łańcuchowej reprezentacji liczby, proszę użyć wbudowanej funkcji str().

Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Proszę po prostu opakować swój kod w wyrażenie funkcyjne.

Oto całe rozwiązanie:

def checkio(num: int) -> str: 

    if not num % 3:
        return "Fizz"
    else:
        return str(num)

Warunek jest zapisany bezpośrednio wewnątrz instrukcji return. 

def checkio(num: int) -> str:
    
    return str(num) if num % 3 else 'Fizz'

Przypadki są traktowane jako elementy krotki, gdzie indeks jest wartością logiczną modulo. 

def checkio(num: int) -> str:

    return ("Fizz", str(num))[bool(num % 3)]
